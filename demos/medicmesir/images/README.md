# Images Folder

Place your images here. The website references these files:

## Required Images

| File | Size | Description | Where Used |
|------|------|-------------|------------|
| `hero-doctor.jpg` | ~1200×800px | Doctor or medical student photo | Homepage hero background |
| `about-main.jpg`  | ~600×750px  | Office/team portrait photo | About section main image |
| `about-students.jpg` | ~400×400px | Students studying together | About section overlay |

## Where to Get Free Images

- **Unsplash** (https://unsplash.com) — Search: "doctor", "medical student", "hospital Malaysia"
- **Pexels** (https://pexels.com) — Free for commercial use
- **Freepik** (https://freepik.com) — Large selection of medical imagery

## Tips

- Portrait-ratio images (taller than wide) work best for `about-main.jpg`
- The hero image uses `mix-blend-mode: luminosity` so it blends with the dark background automatically
- Any JPG or PNG format works — just keep filenames exactly as listed above

If images are missing, the site still looks great with the emoji placeholders!
